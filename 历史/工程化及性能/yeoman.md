# 基本

## 生命钩子

yeoman 生命周期函数执行顺序如下：

- initializing - 初始化函数
- prompting - 接收用户输入阶段
- configuring - 保存配置信息和文件
- default - 执行自定义函数
- writing - 生成项目目录结构阶段
- conflicts - 统一处理冲突，如要生成的文件已经存在是否覆盖等处理
- install - 安装依赖阶段
- end - 生成器结束阶段
